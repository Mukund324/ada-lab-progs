# Ada lab programs

This is a collection of programs done in algorithms lab.

These programs are done in C.

The programs are:

* Breadth-first search

* Depth-first search

* Mergesort

* Quicksort

* Heapsort

* Horsepool's algorithm

* Warshall's algorithm

* Floyd's algorithm

* Knapsack algorithm

* Djistra's algorithm

* Prim's algorithm

* Kruskal's algorithm

* N-queen's method
